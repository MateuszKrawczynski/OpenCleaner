Features in Open Clener:

Customized clean:

- select what kind of junk files you want ot clean up and get rid of them

- you can clean up trash bin, downloads, temporary files, chdisk files, DNS cache, memory dump, notifications, temporary web files, web cookies, web search history, DirectX cache for each graphics card, thumbnail cache, web form data (not recommended, because it will delete all of your autofill saved passwords in the web browser)

Analize files:

- scan your hard drive

- list all your files on disk

- search through all of your files and find the one you are looking for instantly

- check what takes up your disk space the most

Organize files:

- select a folder that you want to organize

- create seperate folders for music, documents, executables and photos, and segregate the files to these folders automatically
